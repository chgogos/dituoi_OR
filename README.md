# Επιχειρησιακή Έρευνα

**setup python environment με το conda**

```
$ conda create --name cplex_community_env python=3.8
$ conda activate cplex_community_env
```

**εγκατάσταση modules**

```
$ pip install cplex
$ pip install docplex
```
